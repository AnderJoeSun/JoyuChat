# JoyuChat
一款用Java开发的类似QQ的聊天软件。11年前的本科毕业设计作品，虽然代码很丑，但每一行因她而很用心:)

_由于历史原因，只留下了源代码，相关的图片资源文件已丢失，不过随便用一些图片代替一下，程序应该可以运行起来_

# 技术
Java, Socket, TCP/IP, MySQL
